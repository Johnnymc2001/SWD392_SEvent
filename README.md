# Technology
- Front-end : ReactJS
- Back-end : ASP.NET
- Mobile : Flutter

- Database : Google Cloud SQL
- Storage : Google Cloud Storage
- Cache Server : Redis Cloud
- Realtime Communication : SignalR
- Google Authentication / Notification : Firebase
